## 🪴 **Introduce**

> **Planto**는 센서 데이터 기반으로 반응하는 스마트 화분을 통해 식물과 상호작용하고, 식물의 상태를 관리할 수 있는 IoT 스마트 화분입니다. 
> 단순 상호작용 뿐만 아니라 사용자와 상호작용을 자신의 감정을 표현하고 성장하는 플랜토를 경험해보세요
